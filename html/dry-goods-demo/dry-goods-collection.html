实在！2019年2月设计圈超实用干货大合集

2月份的设计圈干货来咯！这个月的干货大合集当中涵盖了许多值得收藏的工具和素材，比如梳理得非常清晰的安全检查列表、炫酷的背景生成器，还有定期更新的高质量免费字体素材。还是那句老话，仔细看看吧，总会有一两款正好是你需要的。

（干货来了，前排小板凳准备好！）
DiceBear Avatars

DiceBear Avatars 是一个能够帮你生成炫酷块状头像占位符的工具，你可以通过调用他们的 HTTP API 来生成扁平化的头像。

React Insta Stories

React Insta Stories 是一个可以帮你在网页上生成类似 Instagram 故事或者 Snapchat 故事的组件。使用npm安装它之后，并且使用指向图片URL的数组之后，就可以生成。

Sheety

Sheety 让你可以将任何 Google Sheet 转化为 API，它是完全免费的，你可以将它应用到网站、APP或者其他任何地方。当你在 Google Sheet 中修改任何内容，通过它生成的 API 也都会实时更新。

Minisearch

Minisearch 是一个小巧但是功能强大的全文搜索引擎，适用于 JavaScript。它可以在 Node 和浏览器中轻松运行。

Lobe

Lobe 能够让你的 Web APP 通过视觉化的UI界面进行学习，它能够让 APP 识别笔迹、感知情感甚至听懂音乐。目前 Lobe 还是测试版的，你可以下载 Beta 版尝试一下。

Bubble Toggle

Bubble Toggle 是一款有趣的开关按钮，由 Chris Gannon 所设计。

Textblock

Textblock 是一个 Javascript 工具，可以用来创造更加漂亮的响应式布局。

SocialSizes

SocialSizes 解决了一个非常常见的问题，那就是为图片找到正确的模板，针对不同的社交媒体平台来适配。这款工具包含有 Sketch、PS 和 XD 的模板，并且针对包含 Facebook、Instagram、Linkedin、Twittter、Snapchat、Youtube、Pinterest 和 Twitch。

Awesome Podcast

Awesome Podcast 可以帮你找到你喜欢的播客节目，以便在工作的时候收听。这个列表当中包含了针对不同领域的软件开发工程师的各种播客的汇总。列表已经被组织得很清晰了，你可以从中很快筛选出想要的主题。

Palette App

Palette APP 可以帮你创建更加平滑的配色方案，为了达到这个目标，软件能够让你微调色彩，调整饱和度和渐变的功能非常便捷，你还可以将其他地方的配色方案导入到这当中进行调整。

Photo Creator

Photo Creator 是一款能帮你创建图库的工具，可以帮你混合和调整图库中的角色、背景等元素，最终生成符合你的需求的图片。

Snippet Generator

Snippet Generator 可以让你快速创建并分享代码片段。这款工具还可以在 VSCode、Sublime Text 和 Atom 之间来回切换使用。

SVRF Developers

SVRF Developers 有着世界上最大的 360度的图片和视频素材数据库，使用它的API，你可以为用户提供足够沉浸式的体验，并且API 支持各种类型的APP和使用场景，相机、短信、聊天、创意设计等等。

Can’t Unsee

Can’t Unsee 是一个非常有趣的UI设计相关的游戏，考验你的设计基础知识的时候到了。

JournalBook

JournalBook 是一个私密的、离线的个人日记工具。它提供了完善的引导教程，帮你学习如何记录笔记和想法。所有的数据都存储在本地，你可以试着用用。

Animated Mesh Lines

Animated Mesh Lines 是一个包含有5个 WebGL demo 的 Codrops Playground，它可以帮你了解如何创建有趣的图形样式。

Childhood Flat Icons

Childhood Flat Icons 是一组充满童趣的图标合集，整个合集当中包含有超过100个图标，图标格式为 AI、SVG 和 PNG。

Security Checklist

Security Checklist 是一个非常有用的检查列表，列表中的检查条目能够帮你保护好自己在互联网上的信息安全。

BG-Painter

BG-Painter 是一款颇为有趣的工具，是用来创建动画和静态背景图片用的。你可以从预设的 「Paint」开始着手，然后通过调整达到自己的要求即可。感谢工具的创造者 Frank Hsu。

Static Site Boilerplate

Static Site Boilerplate 是一个用来创造现代静态网站的基础，它包含有你所需要的全部技术基础，你只需要添加代码并部署网站就行了。

UXWing

UXWing 的合集当中可能拥有你所需要的每一个图标，你所需要做的就是搜到你想要的内容并且下载即可。

Startup Illustration Kit

Startup Illustration Kit 是一套用来阐述故事，可视化地呈现事件流程的工具，即使你没有图片素材也可以在它的帮助下，完成展示。整个套件包含30多幅插画，各有特色。

Lindas

Lindas 是一款免费的、可供个人使用的字体，包含大小写字母和数字，作为手写风格字体，Lindas 的风格更加富有力道。

Lorden Holen

Lorden Holen 是一款适合在情人节使用的字体，轻盈，非常容易和爱情联系到一起，个性也很明显。

Reno Mono

Reno Mono 是一款风格现代的等宽字体，它可以在小尺寸的情况下确保可读性。和其他同类字体相比，Reno Mono 显得更有个性也更有特色。

US Blaak

US Blaak 是一款非常有趣的衬线字体，力量感十足，适合展示。每个字母都有着锋利的边缘和笔触。

Venn

Venn 是一款漂亮的字体家族，其中字体有25种不同风格、5种不同的字重。这套字体质量极高，甚至在很多情况下这一套字体就能覆盖从正文到展示性标题等全部的使用。
